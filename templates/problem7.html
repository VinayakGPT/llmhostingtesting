7. Recursive Pattern

Problem Statement:
Given a positive number n, you need to give as an output the following pattern. You need to use recursion. No marks if loops are used.
Example 1:
Input: 11
Output: 11 6 1 -4 1 6 11
Example 2:
Input: 15
Output: 15 10 5 0 5 10 15

            
Incorrect Solution:
#include <stdio.h>
int viper(int b) {
    printf("%d ", b);
    if(b < 0) return b;
    b = b - 5;
    return viper(b);
}
int mountain(int n, int b) {
    b = b + 5;
    printf("%d ", b);
    if(b == n) return 0;
    return mountain(n, b);
}
int main() {
    int n, b;
    scanf("%d", &n);
    b = viper(n);
    mountain(n, b);
    return 0;
}
